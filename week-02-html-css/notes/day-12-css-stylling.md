# CSS Properties Used

## background-color

Changes an element's background.

Example:

```css
body{
    background-color:#f4f4f4;
}
```

---

## color

Changes text color.

```css
h1{
    color:#1f2937;
}
```

---

## padding

Adds space inside an element.

```css
section{
    padding:20px;
}
```

---

## margin

Adds space outside an element.

```css
section{
    margin:20px;
}
```

---

## border

Creates an outline.

```css
img{
    border:3px solid #2563eb;
}
```

---

## border-radius

Rounds corners.

```css
img{
    border-radius:12px;
}
```

---

## Hover Effect

```css
button:hover{
    background-color:#1d4ed8;
}
```
