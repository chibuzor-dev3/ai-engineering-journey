# What is an HTML Table?

An HTML table is used to organize information into rows and columns.

Tables are useful for displaying structured information such as:

- Student grades
- Hospital patient records
- Employee information
- Product inventories
- AI model results
- Learning schedules

Tables begin with the `<table>` element.

---

# HTML Table Elements

## `<table>`

Creates the entire table.

## `<thead>`

Contains the table headings.

## `<tbody>`

Contains the main body of the table where the data is stored.

## `<tr>`

Represents a table row.

## `<th>`

Represents a table heading. It is bold by default and describes the contents of a column.

## `<td>`

Represents table data. It stores the actual information inside each cell.

---

# Difference Between `<th>` and `<td>`

| `<th>` | `<td>` |
|---------|---------|
| Table heading | Table data |
| Bold by default | Normal text |
| Describes a column | Contains the actual data |

# What is an HTML Form?

An HTML form is used to collect information from users.

Forms are commonly used for:

- Login pages
- Registration forms
- Contact forms
- Surveys
- Hospital appointments
- Online shopping
- Job applications

A form begins with the `<form>` element.

---

# HTML Form Elements

## `<label>`

The `<label>` element describes what information should be entered into a form field.

Labels improve accessibility because screen readers can correctly identify each input field.

---

## `<input>`

The `<input>` element allows users to enter information.

Common input types include:

- text
- email
- password
- number
- date
- checkbox
- radio
- file
- color
- range

Each input type is designed for a specific type of information.

---

## `<textarea>`

The `<textarea>` element allows users to enter multiple lines of text.

It is commonly used for:

- Messages
- Comments
- Reviews
- Feedback

Unlike an input field, a textarea can contain long paragraphs.

---

## `<select>`

The `<select>` element creates a dropdown menu where users can choose one option from a list.

Example options:

- Python
- JavaScript
- Java
- C++
- Go

---

## `<option>`

Represents each selectable item inside a dropdown menu.
