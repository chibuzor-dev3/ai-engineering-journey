# Day 15 – JavaScript Fundamentals

## Objective
Learn how to connect JavaScript to an HTML page and display output in the browser console.

## What I Learned

- JavaScript adds interactivity to web pages.
- JavaScript code is usually written in a `.js` file.
- A JavaScript file is linked to an HTML page using:
  ```html
  <script src="script.js"></script>
  ```
- `console.log()` prints messages and values to the browser's developer console.
- Variables store data for use in a program.

## Practice

```javascript
let name = "Chibuzor";
let age = 24;
let isLearning = true;

console.log(name);
console.log(age);
console.log(isLearning);
```

## Key Concepts

- `let` – Declares a variable whose value can change.
- String – Text enclosed in quotes.
- Number – Numeric values.
- Boolean – `true` or `false`.

## Summary

Today I successfully connected JavaScript to my HTML project, created my first JavaScript file, declared variables, and displayed their values in the browser console.
