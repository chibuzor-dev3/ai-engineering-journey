# Day 16 - JavaScript Variables

## What I Learned

- Variables store information.
- `let` creates variables whose values can change.
- `const` creates variables whose values cannot change.
- Variables can store strings, numbers, and booleans.
- Variables can be printed using `console.log()`.
- Variables can be combined to create sentences and perform calculations.

## Practice

let firstName = "Chibuzor";
let age = 24;
let isLearning = true;
