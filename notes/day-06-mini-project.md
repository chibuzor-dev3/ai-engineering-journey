# Day 6 - Mini Project

## Objective

Build a simple patient health checker using JavaScript.

---

## Concepts Used

- Variables
- Data Types
- Functions
- if...else
- console.log()

---

## Why This Matters

Healthcare software rarely uses one concept alone.

Patient information is stored as variables.

Functions process patient data.

Decision statements determine what action should be taken.

All these concepts work together to automate healthcare decisions.

---

## Connection To AI Healthcare

Before AI predicts diseases, software must first collect, organize and evaluate patient information.

Today's project represents the first layer of intelligent healthcare systems.

---

## Reflection
