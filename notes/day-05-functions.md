# Day 5 – Functions

## What I Learned

Today I learned how functions help organize code into reusable blocks.

Instead of repeating code multiple times, I can create one function and call it whenever I need it.

## HealthTech Connection

Hospital systems perform many repetitive tasks:

- Registering patients
- Checking temperatures
- Calculating BMI
- Scheduling appointments

Functions make these tasks reusable and easier to maintain.

## Key Takeaways

- Functions reduce repetition.
- Parameters let functions work with different data.
- Return values send information back.
- Functions are essential for building larger applications.
