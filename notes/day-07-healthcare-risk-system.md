# Project Overview

The Healthcare Risk Assessment System is a simple JavaScript program that demonstrates how software can automate basic healthcare assessments.

The program:

- Stores patient information.
- Calculates Body Mass Index (BMI).
- Checks for fever.
- Determines whether the patient is an adult or a pediatric patient.
- Classifies BMI.
- Produces an overall health recommendation.

Although this is a beginner project, it models the logic found in real healthcare software.

---

# How the Program Works

## Step 1: Store Patient Information

The program stores patient details using variables.

Example:

- Patient Name
- Age
- Weight
- Height
- Temperature

These variables act as the input for the program.

---

## Step 2: Calculate BMI

A reusable function calculates the patient's Body Mass Index using weight and height.

Instead of repeating the formula throughout the program, the function performs the calculation whenever it is needed.

---

## Step 3: Evaluate Patient Information

The application uses conditional statements to determine:

- Whether the patient has a fever.
- Whether the patient is an adult or pediatric patient.
- Whether the BMI is underweight, healthy, overweight, or obese.

---

## Step 4: Overall Risk Assessment

The program combines multiple conditions using the logical AND operator (`&&`).

For example:

- High temperature AND high BMI → High Risk
- Otherwise → Low Risk

This demonstrates how software combines multiple pieces of information before making a recommendation.
