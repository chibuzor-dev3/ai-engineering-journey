Today I learned about the basic components that make a computer work:

- RAM
- Storage

Understanding these foundations is important because every software application depends on these components.


## CPU (Central Processing Unit)

The CPU is the part of the computer responsible for executing instructions.

It processes commands from programs and performs calculations that allow applications to run.

Example:
When I open VS Code, the CPU helps process the instructions needed to launch and run the application.


## RAM (Random Access Memory)

RAM - Random Access Memmory. Is used to store relatively small amounts of data that the CPU and GPU need to be able to access quickly so that everything on your computer rens smoothly.

RAM is simply a type of storage.

Example:
When I open VS Code and Chrome at the same time, both applications use RAM.

RAM is temporary, meaning data is removed when the computer is turned off.

---

## Storage

Storage is where files and programs are permanently saved.

Examples:
- Documents
- Applications
- Images
- Videos

Types:
- HDD
- SSD

SSD is faster because it uses electronic memory instead of moving parts.

---

## Operating System

An operating system is a program that manages the computer hardware.

It also provides a basis for application programs and acts as an intermediary between computer user and computer hardware

Examples:
- Windows
- Linux
- macOS

The operating system allows applications to run and manages resources like memory and storage.
